# **App Name**: PomoVNO

## Core Features:

- Timer Functionality: Implements a fully functional pomodoro timer with customizable work and break durations, including standard 25-minute work and 5-minute short break intervals, and a 15-minute long break every four sessions.
- Session Management: Keeps track of the pomodoro session count to trigger the long break automatically after every four sessions.
- Auditory Notifications: Plays a sound at the start and end of each work and break session, with an option to toggle the sound on or off in settings.
- Visual Cues: Flashes the screen five times at the start and end of each session to provide a clear visual indication of session changes.
- Customizable Durations: Allows users to customize the duration of work, short break, and long break intervals according to their preferences.
- Mobile Responsiveness: Ensures the application is fully responsive and optimized for both mobile and web environments, providing a consistent experience across devices.
- Full Screen Mode: Enables a full-screen mode, particularly suited for mobile devices, to minimize distractions and provide an immersive timing experience.

## Style Guidelines:

- Primary color: Pure white (#FFFFFF) for all UI elements to stand out against the background.
- Background color: Complete black (#000000) for a minimalistic and distraction-free environment.
- Font: 'Inter', a sans-serif font, for both headlines and body text, chosen for its clean and modern appearance in a minimalistic UI.
- Simple, geometric icons in white to represent timer controls (start, pause, reset) and settings.
- Clean and centered layout with large, easily readable timer display.
- Smooth, subtle transitions for timer updates and screen flashes to avoid distractions.
